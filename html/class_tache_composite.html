Detailed Description

Classe permettant de manipuler des TacheComposite. Les Projet instancient cette classe, pas l'utilisateur directement. Une TacheComposite n'est pas programmable mais peut contenir des sous-tâches, TacheUnitaire comme TacheComposite.
