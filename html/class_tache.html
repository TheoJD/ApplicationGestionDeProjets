Detailed Description

Classe permettant de manipuler des tâches. Cette classe est abstraite, on ne l'instanciera que par l'intermédiaire de ses classes filles. Les dates doivent être cohérentes, une tâche doit appartenir à un Projet, mais peut ne pas avoir de TacheComposite mère.
